# Flask To-Do App

A simple to-do list app with user login and registration, built with Flask and SQLite.

## Setup

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python run.py
```

Open [http://127.0.0.1:5000](http://127.0.0.1:5000) in your browser.

## Features

- Register and log in
- Add, update, and clear tasks
- SQLite database (`instance/todo.db` is created on first run)
